# Hotel Reservation System (Java)

## Project Description
This project is a Java-based Hotel Reservation System that allows users to search, book, and manage hotel rooms efficiently. It is developed using Object-Oriented Programming (OOP) concepts and includes file handling for storing booking details.

---

## Features
- View available rooms
- Room categorization (Standard, Deluxe, Suite)
- Book hotel rooms
- Cancel reservations
- View booking details
- Payment simulation
- File handling to store booking data

---

## Technologies Used
- Java
- OOP Concepts (Classes & Objects)
- ArrayList
- File Handling

---

## How to Run

1. Compile the program:
   javac HotelReservationSystem.java

2. Run the program:
   java HotelReservationSystem

---

## System Workflow
1. Display available rooms
2. User selects a room to book
3. Payment is simulated
4. Booking details are stored
5. User can cancel or view bookings
